# sweetcrumbs-bakery

A pure HTML bakery website created as part of a web development assignment.

## Features

* Top utility bar
* Header and logo section
* Navigation bar
* Hero/banner section
* Announcement strip
* About Us section
* Menu section
* Weekly specials section
* Gallery section
* Testimonials section
* Contact form
* Location section
* Four-column footer

## Technologies Used

* HTML only
* Tables for layout
* Native HTML forms
* Internal anchor navigation
